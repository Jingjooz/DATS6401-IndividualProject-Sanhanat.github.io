ABOUT THE TOOLS

This data visualization project was prepared for DATS 6401: Visualization of Complex, taught by Professor Nima Zahadat. HTML5, CSS, and JAVASCRIPT were used to work together for presenting the government spending of G20 countries. Moreover, the code of some part in this project was adapted elements from W3 School, Google visualization chart for developer, and other online sources. In addition, Adobe Photoshop CC 2019 was used to edit the images that were in this project.

ABOUT THE DATA

To create data visualization, we used the G20 spending data, collected yearly from 2012-2016, from World Bank. RStudio and Microsoft Excel were used to clean the data before containing them into Google Excel Speadsheets.
